UserSettingsViewController

class UserSettingsViewController: UIViewController

This Vc handles all the communication to display value for user settings. It guides the user througth new Vcs : language, location and currency settings. For sanity, this controller does set any value. All the value are saved in the dedicated vcs throuth dedicated models
